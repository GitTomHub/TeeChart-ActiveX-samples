The compiled version of the
TeeChart2017VBCtl.ocx control is
installed in the TeeChart root
folder at install time to 
support TeeChart's ASP 
Usercontrol example. if you 
wish to make changes and see 
their effect, be sure to 
register your compiled ocx.

eg.
regsvr32 c:\yourfolder\TeeChart2017VBCtl.ocx
